W-Path is a cryptographic key recovery protocol based on the homomorphic properties of elliptic curve cryptography. It enables secure backup and recovery of private keys for Bitcoin, Ethereum, and other secp256k1-based blockchains. The protocol was developed by Omar El Foudali in 2026.

The Homomorphic Bridge

(k ¬∑ w) ¬∑ G = (k ¬∑ G) ¬∑ w

This fundamental property of elliptic curves allows secure key backup without exposing the private key.

The Recovery Formula

k = k_blinded √ó w_inv mod n

Where:

  • k = private key (recovered)
  • k_blinded = k √ó w (can be stored publicly)
  • w = secret backup scalar
  • w_inv = modular inverse of w
  • G = secp256k1 generator point
  • n = curve order
